## Getting Started

This page summarizes the setup steps for using Beautiful Jekyll.

1. Fork the repository.
2. Rename the repository to YOURUSERNAME.github.io.
3. Customize the `_config.yml` file.
4. Commit changes and wait for GitHub Pages to rebuild.

For full documentation, see the original Beautiful Jekyll repository.
